RaspiBlitz

Your DIY Bitcoin & Lightning Node on a RaspberryPi

RaspiBlitz is an open-source project that allows you to run your own Bitcoin and Lightning Network node on a Raspberry Pi.

Easy Installation

Download pre-built SD card images for quick and easy setup on your Raspberry Pi.

Full Bitcoin Node

Operate a full Bitcoin node to verify the blockchain and transactions yourself.

Lightning Network

Run a Lightning Node to send and receive fast, low-cost Bitcoin transactions.

Additional Apps

Install and access additional apps and services directly from RaspiBlitz node.

Open Source

All parts of RaspiBlitz are MIT licensed, and thus fully open source.

Buy RaspiBlitz pre-built or Build yourself

You can build RaspiBlitz yourself or buy a pre-built RaspiBlitz from a third party.
